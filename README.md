Secure Password Generator

Description
SecurePassGen is a Python-based password generator that creates strong and secure passwords. It ensures that every password contains at least one uppercase letter, one lowercase letter, one digit, and one symbol. You can generate multiple passwords at once, making it ideal for managing accounts securely.

This project is beginner-friendly and perfect for anyone looking to improve their online security with strong, random passwords.

Features
- Generates strong passwords of customizable length
- Includes uppercase, lowercase, digits, and symbols
- Generates multiple passwords in one go
- Warns if the password length is too weak (less than 8 characters)
- Easy-to-use, beginner-friendly interface
